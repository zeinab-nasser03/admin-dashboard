# Admin Dashboard

A modern and responsive **Admin Dashboard** built using **HTML, CSS, and JavaScript**. This project was developed as part of a Web Development Internship Task to demonstrate JavaScript fundamentals, DOM manipulation, event handling, and form validation.


## Features

### User Interface

* Responsive dashboard layout
* Modern sidebar navigation
* Statistics cards (Sales, Expenses, Income)
* Recent updates section
* Sales analytics panel
* Mobile-friendly design

### JavaScript Functionality

* Dark/Light mode toggle
* Responsive sidebar menu
* Product management system
* Add Product modal popup
* Dynamic product table updates
* Product search functionality
* Delete product functionality
* LocalStorage data persistence

### Form Validation

* Required field validation
* Product price validation
* Prevents invalid submissions


## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Font Awesome
* LocalStorage API
